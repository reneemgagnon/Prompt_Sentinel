# Plugin Usage

This Claude package bundles:

- `SKILL.md` at the package root for skill discovery
- `agents/prompt-sentinel-guardian.md` for guarded specialist behavior
- `hooks/hooks.json` plus Python hook scripts for deterministic enforcement
- `.claude-plugin/plugin.json` for plugin packaging

The skill helps Claude reason about guarded workflows.
The hooks enforce pre-tool and tamper-sensitive behavior.
