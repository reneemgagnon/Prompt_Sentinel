# prompt-sentinel-core

Installable local runtime, CLI, and SDK for Prompt_Sentinel.

## Planned responsibilities

- load sealed policies
- evaluate untrusted tool proposals
- issue and verify signed capability tickets
- keep tamper-evident audit logs
- expose adapters for agent-specific integrations

## Current state

This is a scaffold with starter implementations and extension points. The repo's
working V3 prototype should be migrated into this package module by module.
