# Operating Modes

## Local developer mode

- use a local policy file
- use local signing keys
- log to local JSONL
- alert locally on suspicious instruction or config changes

## Team-managed mode

- use distributed policy bundles
- use signed approval workflows
- export audit and alert records centrally

## Enterprise mode

- central policy distribution
- managed keys
- approval service
- SIEM and compliance export
- shared sanitized threat vectors
